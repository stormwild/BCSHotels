# BCSHotels

Sample Test Application for BCSTechnology

## Local Development

### Pre-requisite

- SQL Server Express with LocalDb

### Setup

Build the solution

Run the web project

Using the package manager console, run:

```sh
cd BCSHotelsDomain
Update-Database
```

Rerun the web application
